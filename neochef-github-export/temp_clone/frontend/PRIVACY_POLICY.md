## 3. Données Collectées

### 3.1 Données fournies par l'utilisateur
- **Informations de compte** : nom, prénom, adresse email
- **Informations professionnelles** : rôle dans le restaurant, catégories assignées
- **Données de réservation** : nom du client, nombre de personnes, date et heure

### 3.2 Données collectées automatiquement
- **Données d'utilisation** : pages visitées, fonctionnalités utilisées
- **Données techniques** : type d'appareil, système d'exploitation, identifiant unique

### 3.3 Données NON collectées
- Nous ne collectons PAS de données de localisation
- Nous ne collectons PAS de données de santé
- Nous ne collectons PAS de données financières personnelles

## 4. Utilisation des Données

Nous utilisons vos données pour :
- Fournir et améliorer nos services
- Gérer votre compte utilisateur
- Communiquer avec vous concernant le service
- Assurer la sécurité de l'application

## 5. Partage des Données

Vos données ne sont PAS vendues à des tiers.

Nous pouvons partager vos données avec :
- **Hébergeurs** : pour le stockage sécurisé des données
- **Services d'analyse** : pour améliorer l'application (données anonymisées)

## 6. Sécurité des Données

Nous mettons en œuvre des mesures de sécurité appropriées :
- Chiffrement des données en transit (HTTPS)
- Accès restreint aux données personnelles
- Mots de passe hashés

## 7. Conservation des Données

- **Données de compte** : conservées tant que le compte est actif
- **Données de réservation** : conservées 3 ans pour obligations légales
- **Données techniques** : conservées 12 mois maximum
